Descript.js
======================
  Descript.js can parse descript.txt file.

Demo
----------
* https://dl.dropbox.com/u/265158/GitHub/descript.js/index.html

Usage
----------
    var hash = new Descript(Text);
    console.dir(hash);

Features
----------------
* parsing descript.txt to Object
